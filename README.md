# ts-cours-map

## Getting Started

- стягиваем проект

git clone git@github.com:AnnSyh/ts-cours.git

 - переходим в папку ts-cours/

cd ts-cours/

- устанавливаем зависимости

npm i

- переходим в папку src/

cd src/

 - и запускаем в ней проект 

npm run build

npm run dev

- Запускается localhost:

Server running at http://localhost:1234
